Helpful Ruby Methods

  • .each: this iterates over an array. Syntax: array.each do {|x| block of code}
  • .each_with_index: this iterates of an array and also produces the index of that array. Syntax: array.each_with_index do {|value, index| block of code}
  • .split(): this splits a string up into an array. Syntax: string.split(" "). The parenthesis indicate what the divider will be.
  • .join(): this joins an array back into a string, the same way split breaks it apart.
  • .reverse: this reverses an array. Syntax: array.reverse.
