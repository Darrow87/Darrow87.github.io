Hey guys, this will be a blog about my Dev Bootcamp experience. I'm a little tired now so I'll just answer some basic questions based on the assignment that I created today.

What are the benefits of version control? Well, this is literally my first day in the programming world, so this will be theoretical. But basically version control is good because it allows many people to make changes without overwriting one another. It also allows you to go back to a previous saved version if you accidentally implement a version that is buggy.

How does git help you keep track of changes? Well, when you make an official change in git it is called a commit. You can then look at previous commits to see what you changed and when. There is (ideally) a short description provided for each commit so you can easily tell what changes were made.

Why use GitHub to store your code? I'm not exactly sure what the other options are, however, it has been easier than I would have thought to get a basic website up and running, so that is a definite plus.
