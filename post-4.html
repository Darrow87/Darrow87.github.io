Hey there, this post will be about the enumerable method group_by

Essentially, the group_by method groups the elements of an array into a hash based on a certain criteria. For example, say you wanted to group an array of animal names by their length. array = ["lion", "tiger", "bear", "rhino"]. grouped_array = array.group_by {|animal| animal.length}. This would return a hash. grouped_array = {4 => [lion, bear], 5 => [tiger, rhino]}.

As you can see the criteria that we passed, animal.length, became the key for the hash, and the items of the array that satified that criteria became the values for that hash, in an array format.
